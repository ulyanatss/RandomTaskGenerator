# Random Task Generator

**Автор:** Ульяна Цыпышева

## Описание
Приложение с графическим интерфейсом (Python + Tkinter) для генерации случайных задач. Позволяет:
- Генерировать случайную задачу из предопределённого или пользовательского списка.
- Сохранять историю сгенерированных задач в JSON-файл (автоматическая загрузка при запуске).
- Фильтровать отображаемую историю по категориям (учёба, спорт, работа).
- Добавлять новые задачи с указанием категории (проверка на пустую строку).
- Очищать историю.

## Требования
- Python 3.6+
- Стандартные библиотеки: tkinter, random, json, os
